In this project we are accepting the file name from the user 
and writing into the file 

(Note: create a file first so we can write into it)

STEPS:
1.To build the project type: make build

2.To run the file type: make ARGS="filename" run
	eg: make ARGS="test.txt" run
  Enter the text you want to write into the file 

3.To remove the generated executable file type: make clean

4.To get some help type: make help
