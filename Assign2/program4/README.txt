stat is a built in structure
and we use inbuilt macros to see the file permisssions in the given program 

(Note: The file you are gaining information about should be created first in the current working directory)

Steps to run the following code:

1.create the file to want info about the file in currect working directory

2.to open the file type: make build

3.to gain the information about the file type: make ARGS="filename" info

4.to delete the executable we created type: make clean
