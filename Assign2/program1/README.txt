In this project we are implementing to create a file and accept its name from user and open that file

1.to create a file type: make create

2.to run the created file type: make ARGS="filename" runcreate
(Note:the filename should be included within the double quotes)

3.to open the file type: make open

4.to run the created file we just created type: make ARGS="filename" runopen
(Note:the filename should be included within the double quotes)

5.to delete the exe file we created type: make clean
